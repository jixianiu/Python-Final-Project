# Python-Final-Project
 2025 秋《基于 Python 的问题解析》期末大作业
## 项目简介
本项目聚焦全球公共健康问题 —— 肥胖，通过数据挖掘与机器学习技术，分析肥胖的核心影响因素并构建预测模型。
项目基于多维度肥胖数据集，完成数据预处理、探索性数据分析、多模型对比建模等工作，最终实现对个体肥胖水平的精准预测，同时识别关键影响因素，为肥胖预防与健康干预提供数据支撑和科学依据。
## 数据来源
肥胖风险数据集 - Heywhale.com
